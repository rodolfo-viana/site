Controle de fluxo com if¶

Até o momento, estamos escrevendo códigos que são executados numa estrutura fixa: de cima para baixo, linha a linha. Exemplo:

In [1]:
nome = input("Digite seu nome: ")
ano = input("Digite o ano de nascimento: ")
idade = 2023 - int(ano)
print("Bom dia, {}. Você tem ou terá {} anos em 2023.".format(nome, str(idade)))

Digite seu nome: Rodolfo
Digite o ano de nascimento: 1981
Bom dia, Rodolfo. Você tem ou terá 42 anos em 2023.

Até aqui, nosso script processa a linha 1, depois a linha 2, depois a linha 3 e, por fim, a linha 4. Uma após a outra, sem nunca pular uma linha. É um fluxo fixo.

Mas podemos mudar isso. Podemos controlar a execução, o fluxo do script. Uma das formas é com if, que condiciona a execução de uma linha ao resultado de outra.

if¶

A estrutura mais simples do controle de fluxo com if é assim:

if condicao_1:
    operacao_1 # reparem que há quatro espaços antes de `operacao_1`

Ou seja, em português seria algo como "se a condição 1 for True, realize a operação 1; se não, não realize".

Desenhando,

Vamos ver isso na prática:

In [2]:
salario = 1302 * 2 # dois salários mínimos
aluguel = 1600
luz = 90
agua = 80
telefone = 99
cartao = 200
contas = aluguel + luz + agua + telefone + cartao

print("Começou o fluxo.")

if salario >= contas: # se o salário for maior ou igual ao valor das contas...
    restos = salario - contas # ...calcule quanto sobra...
    print("Consigo pagar as contas e me sobram {} reais.".format(restos)) # ...e imprima a frase com o valor
    
print("Acabou o fluxo.")

Começou o fluxo.
Consigo pagar as contas e me sobram 535 reais.
Acabou o fluxo.

No exemplo acima o cálculo de restos foi executado e a frase foi impressa porque salario > contas é True. Vejamos o que acontece se o salário for menor.

if-else¶

Quando utilizamos if, a operação é executada apenas se a condição for True. Contudo, há casos em que, mesmo dando False queremos executar uma operação alternativa. Para isso usamos if-else. Funciona assim:

if condicao_1:
    operacao_1
else:
    operacao_2

Ou seja, em português seria algo como "se a condição 1 for True, realize a operação 1; se não, realize a operação 2".

Desenhando,

Vejamos um exemplo — primeiro, com dois salários; depois, com um salário:

In [4]:
salario = 1302 * 2
aluguel = 1600
luz = 90
agua = 80
telefone = 99
cartao = 200

    restos = salario - contas # ...calcule quanto sobra...
    print("Consigo pagar as contas e me sobram {} reais.".format(restos)) # ...e imprima a frase com o valor
else: # se não...
    divida = contas - salario # calcule a diferença...
    print("Não consigo pagar as contas. Minha dívida é de {} reais.".format(divida)) # ...e imprima a frase com o valor

if-elif-else¶

Vimos que o uso de if-else permite executar uma operação para resultado True e outra, distinta, para False. Mas e se tivermos três ou mais operações? Por exemplo:

(1) Se a temperatura estiver acima de 30°C, devo usar protetor solar; (2) se estiver entre 20°C e 30°C, basta uma camiseta; (3) se estiver entre 10°C e 19°C, devo usar uma blusa; (4) se estiver abaixo de 10°C, preciso de jaqueta.

Repare que, na construção acima, tenho 4 condições que levam a 4 operações distintas:

condição operação
> 30 protetor solar
20-30 camiseta
10-19 blusa
< 10 jaqueta

Como ficaria isso em Python? Bem, teríamos de ir além de if-else. Temos de usar if-elif-else:

if condicao_1:
    operacao_1
elif condicao_2:
    operacao_2
elif condicao_3:
    operacao_3
...
elif condicao_n:
    operacao_n
else:
    operacao_z

Ou seja, em português seria algo como "se a condição 1 for True, realize a operação 1; se não, veja de satisfaz a condição 2 e, se satisfazer, realize a operação 2; se não, veja de satisfaz a condição 3 e, se satisfazer, realize a operação 3 (...), se nenhuma condição for satisfeita, realize a operação z".

Desenhando,

O exemplo das temperaturas em Python ficaria assim:

In [6]:
temperatura = 28
if temperatura > 30:
    print("Devo usar protetor solar")
elif temperatura <= 30 and temperatura >= 20:
    print("Devo usar apenas uma camiseta")
elif temperatura <= 19 and temperatura >= 10: 
    print("Devo usar uma blusa")
else:
    print("Devo usar uma jaqueta")
# Repare que coloquei a última condição (temperatura < 10) no 
# `else`, pois é a única condição que resta dentre todas. 
# Ou seja, a condição fica implícita.

Controle de fluxo com while¶

Vimos que o fluxo de um script pode ser controlado, que o código nem sempre precisa ter todas as suas linhas lidas. E vimos que, uma forma de controlar o fluxo, é com if.

Há outra forma: while. Diferentemente de if, que checa se a condição é True ou False, while executa a operação enquanto a condição for True. Sua sintaxe é assim:

while condicao:
    operacao

Um exemplo concreto:

Tenho um micro-ônibus com nenhum passageiro e capacidade para 20 pessoas. Enquanto a lotação for menor que 20, adiciono passageiro.

Em Python, ficaria assim:

In [8]:
passageiros = 0
lotacao = 20
while passageiros < lotacao:
    passageiros = passageiros + 1
    print("Tenho {} passageiro(s) no meu ônibus.".format(passageiros))
