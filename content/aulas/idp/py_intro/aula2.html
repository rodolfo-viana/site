Controle de fluxo com if¶

At o momento, estamos escrevendo cdigos que so executados numa estrutura fixa: de cima para baixo, linha a linha. Exemplo:

In [1]:
nome = input("Digite seu nome: ")
ano = input("Digite o ano de nascimento: ")
idade = 2023 - int(ano)
print("Bom dia, {}. Voc tem ou ter {} anos em 2023.".format(nome, str(idade)))

Digite seu nome: Rodolfo
Digite o ano de nascimento: 1981
Bom dia, Rodolfo. Voc tem ou ter 42 anos em 2023.

At aqui, nosso script processa a linha 1, depois a linha 2, depois a linha 3 e, por fim, a linha 4. Uma aps a outra, sem nunca pular uma linha. um fluxo fixo.

Mas podemos mudar isso. Podemos controlar a execuo, o fluxo do script. Uma das formas com if, que condiciona a execuo de uma linha ao resultado de outra.

if¶

A estrutura mais simples do controle de fluxo com if assim:

if condicao_1:
    operacao_1 # reparem que h quatro espaos antes de `operacao_1`

Ou seja, em portugus seria algo como "se a condio 1 for True, realize a operao 1; se no, no realize".

Desenhando,

Vamos ver isso na prtica:

if-else¶

Quando utilizamos if, a operao executada apenas se a condio for True. Contudo, h casos em que, mesmo dando False queremos executar uma operao alternativa. Para isso usamos if-else. Funciona assim:

if condicao_1:
    operacao_1
else:
    operacao_2

Ou seja, em portugus seria algo como "se a condio 1 for True, realize a operao 1; se no, realize a operao 2".

Desenhando,

Vejamos um exemplo primeiro, com dois salrios; depois, com um salrio:

if-elif-else¶

Vimos que o uso de if-else permite executar uma operao para resultado True e outra, distinta, para False. Mas e se tivermos trs ou mais operaes? Por exemplo:

(1) Se a temperatura estiver acima de 30C, devo usar protetor solar; (2) se estiver entre 20C e 30C, basta uma camiseta; (3) se estiver entre 10C e 19C, devo usar uma blusa; (4) se estiver abaixo de 10C, preciso de jaqueta.

Repare que, na construo acima, tenho 4 condies que levam a 4 operaes distintas:

condio operao
> 30 protetor solar
20-30 camiseta
10-19 blusa
< 10 jaqueta

Como ficaria isso em Python? Bem, teramos de ir alm de if-else. Temos de usar if-elif-else:

if condicao_1:
    operacao_1
elif condicao_2:
    operacao_2
elif condicao_3:
    operacao_3
...
elif condicao_n:
    operacao_n
else:
    operacao_z

Ou seja, em portugus seria algo como "se a condio 1 for True, realize a operao 1; se no, veja de satisfaz a condio 2 e, se satisfazer, realize a operao 2; se no, veja de satisfaz a condio 3 e, se satisfazer, realize a operao 3 (...), se nenhuma condio for satisfeita, realize a operao z".

Desenhando,

O exemplo das temperaturas em Python ficaria assim:

In [6]:
temperatura = 28
if temperatura > 30:
    print("Devo usar protetor solar")
elif temperatura <= 30 and temperatura >= 20:
    print("Devo usar apenas uma camiseta")
elif temperatura <= 19 and temperatura >= 10: 
    print("Devo usar uma blusa")
else:
    print("Devo usar uma jaqueta")
# Repare que coloquei a ltima condio (temperatura < 10) no 
# `else`, pois  a nica condio que resta dentre todas. 
# Ou seja, a condio fica implcita.

Controle de fluxo com while¶

Vimos que o fluxo de um script pode ser controlado, que o cdigo nem sempre precisa ter todas as suas linhas lidas. E vimos que, uma forma de controlar o fluxo, com if.

H outra forma: while. Diferentemente de if, que checa se a condio True ou False, while executa a operao enquanto a condio for True. Sua sintaxe assim:

while condicao:
    operacao

Um exemplo concreto:

Tenho um micro-nibus com nenhum passageiro e capacidade para 20 pessoas. Enquanto a lotao for menor que 20, adiciono passageiro.

Em Python, ficaria assim:

In [8]:
passageiros = 0
lotacao = 20
while passageiros < lotacao:
    passageiros = passageiros + 1
    print("Tenho {} passageiro(s) no meu nibus.".format(passageiros))
